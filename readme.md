# LocateHub Releases

This repository contains public release builds for **LocateHub**.

LocateHub is a desktop utility locate tracking application designed around **Click Before You Dig Manitoba** locate tickets and **Manitoba excavation / dig permit workflows**.

It helps organize:

- Jobs
- Locate tickets
- Dig permits
- Utility/station responses
- Missing response tracking
- PDF packages
- Job file storage
- Backups and updates

## What is included here

This repository is for release files only, such as:

- LocateHub update ZIP files
- LocateHub installer EXE files
- Release notes

The source code is not published in this repository.

## Release files

Typical release assets include:

- `LocateHub_#.#.zip`  
  Used by the in-app updater.

- `LocateHub_Setup_#.#.exe`  
  Used for first-time installation.

## Intended workflow

LocateHub is currently designed primarily for:

- Click Before You Dig Manitoba ticket PDFs
- Manitoba locate response workflows
- Manitoba excavation / dig permit tracking
- Local desktop use on Windows

It may not correctly parse or manage locate tickets, permits, or utility response formats from other provinces, regions, or locate systems without additional testing and changes.

## Recommended use

Before updating or installing:

1. Back up your current LocateHub data.
2. Confirm the release version is newer than your installed version.
3. Use the installer for new installations.
4. Use the ZIP package for in-app/manual updates.
5. Confirm your tickets, responses, permits, packages, and job files are correct after updating.

## Data safety

LocateHub stores working data locally on the user's machine, including job data, tickets, packages, logs, and backups.

Users are responsible for maintaining their own backups.
